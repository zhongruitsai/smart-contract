一、委託投票（Proxy Voting）

功能說明

股東可將投票權委託給另一位股東代為行使。委託後原股東無法自行投票，代理人可代替委託人投票。代理人投票後委託無法撤回。

合約層（src/GovernanceVoting.sol）

功能 函數 行號
儲存委託關係的 mapping proxyOf 第 129 行
記錄代理人是否已投票 proxyHasVoted 第 130 行
授予委託 grantProxy(proposalId, proxy) 第 298–306 行
代理人代為投票 voteOnBehalf(proposalId, delegator, forVotes, againstVotes, abstainVotes) 第 309–323 行
撤回委託（代理人未投票前） revokeProxy(proposalId) 第 326–330 行

關鍵邏輯： - 第 304 行：一個提案只能委託一次，ProxyAlreadyGranted 錯誤防止重複委託 - 第 313 行：只有被委託的代理人才能呼叫 voteOnBehalf，否則拋出 NotAProxy - 第 328 行：代理人已投票後呼叫 revokeProxy 會拋出 CannotRevokeAfterProxyVoted

二、聯署提案（Cosign Proposal）

功能說明

股東可發起「聯署提案」，需集滿 10 位不同股東連署，且所有連署人合計持股達總股份 1% 以上，提案才正式生效。連署截止日等同提案期截止日。

關鍵邏輯： - 第 252 行：連署截止後呼叫 cosign 會拋出 CosignDeadlinePassed - 第 253 行：同一人不能連署兩次，拋出 AlreadyCosigned - 第 260–263 行：第 10 位連署人成功後，自動呼叫持股驗證並將 isActive 設為 true

三、時間快轉（Time Warp）

功能說明

為了 Demo 方便，管理員可在合約層面快轉時間，模擬提案期、投票期截止等流程，而不需要真的等待數週。

合約層

兩個合約（GovernanceVoting.sol、DirectorElection.sol）都有相同的時間偏移機制：

功能 行號（兩合約相同）
儲存時間偏移量 uint256 public timeOffset
取得目前合約時間 function _now() → block.timestamp + timeOffset
對外查詢合約時間 function currentTime()
累加時間偏移（僅管理員） function addTimeOffset(uint256 secs)

GovernanceVoting.sol 對應行號：第 119–123 行 DirectorElection.sol 對應行號：第 86–90 行

前端層（frontend/components/TimeController.tsx）

功能 行號
讀取合約目前時間 第 51–57 行
本地網路（Anvil）快轉 第 13–24 行：呼叫 evm_increaseTime + evm_mine
Sepolia 快轉 第 68–79 行：呼叫兩個合約的 addTimeOffset
預設快轉按鈕（+1天 到 +60天） 第 34–42 行
自訂天數輸入 第 90–94 行
